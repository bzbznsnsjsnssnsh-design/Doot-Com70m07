===============================================================
   الجزء 1: نظرة عامة وهيكل المشروع
===============================================================

هذا المشروع هو "مترجم الفيديو" - تطبيق ويب يترجم فيديوهات يوتيوب إلى العربية
مع توليد صوت عربي متزامن مع الفيديو الأصلي.

هيكل المشروع (Monorepo):
===============================================================

المشروع/
├─ lib/
│   ├─ api-spec/          ← تعريف واجهة البرمجة (OpenAPI)
│   ├─ api-zod/           ← أنماط التحقق من البيانات
│   ├─ api-client-react/  ← أوامر React Query جاهزة
│   └─ db/                ← قاعدة البيانات (PostgreSQL + Drizzle)
│
├─ artifacts/
│   ├─ video-translator/  ← الواجهة الأمامية (ما تراه كمستخدم)
│   ├─ api-server/        ← الخادم الخلفي (المعالج)
│   └─ mockup-sandbox/    ← بيئة تطوير المكونات البصرية
│
├─ scripts/               ← سكربتات مساعدة
├─ pnpm-workspace.yaml    ← تعريف حزم workspace
└─ package.json           ← الأوامر الرئيسية

التقنيات المستخدمة:
===============================================================
• Frontend: React 19, Vite 7, Tailwind CSS 4, Framer Motion, Wouter
• Backend: Express.js, Node.js 24, Pino (logging)
• AI/ML: faster-whisper (Python), OpenAI API, Microsoft Edge TTS
• Multimedia: ffmpeg (معالجة الصوت), yt-dlp (تحميل من يوتيوب)
• Database: PostgreSQL + Drizzle ORM
• Monorepo: PNPM Workspaces

هذا الملف يُعرّف كل نقاط API التي يقدمها الخادم.
يستخدم أداة Orval لتوليد الكود تلقائياً من هذا الملف.

نقاط API المعرفة:
  GET  /healthz              - فحص صحة الخادم
  POST /translate/process     - بدء معالجة مقطع فيديو
  GET  /translate/status/:id - الاستعلام عن حالة المهمة
  GET  /translate/audio/:id  - جلب ملف الصوت
  GET  /translate/models     - قائمة الأصوات المتاحة
  GET  /translate/info?url=  - معلومات الفيديو (العنوان)
  GET  /translate/cookies/status
  POST /translate/cookies      - حفظ كوكيز
  DELETE /translate/cookies   - حذف كوكيز

الأنماط (Schemas):
  ProcessVideoRequest:
    - videoUrl: string          (رابط يوتيوب)
    - startTime: number          (وقت البداية بالثواني)
    - voice: string             (اسم صوت Edge TTS)
    - translationEngine: enum   ("openai" | "google" | "pollinations")
    - forceAudioExtraction: boolean

  JobStatusResponse:
    - jobId, status, progress
    - audioUrl, transcript, translation
    - error, startTime, suggestedRate
    - audioDurationSec
    - sentences: []              (مصفوفة الجمل مع التوقيتات)

  TtsVoice:
    - id, name, lang            (14 صوت عربي + متعدد اللغات)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━ loop التزامن الرئيسي (100ms) ━━━━━━━━━━━━━━━━━━━━━━━━━━━━

startSyncLoop():
  يُشغّل loop كل 100ms يفعل:

  1. قراءة وقت الفيديو الحالي
  2. كشف التقديم (seek): إذا قفز >3s → إعادة ضبط كل شيء
  3. تحديد المقطع الحالي: seg = getSegStart(videoTime, step)
  4. جلب المهمة: job = jobsRef.current.get(seg)
  5. إيجاد الجملة الحالية: sentenceIdx = findIndex(relTime)

  المنطق لكل حالة:
  - جملة معلّقة (pending): تشغيلها عند اكتمالها
  - الفيديو تقدم +2 جمل: catch-up → تشغيل الجملة الحالية
  - لا شيء يعمل + جملة جاهزة: تشغيلها
  - توقف مؤقت: استئناف التشغيل
  - تعديل سرعة الصوت: playbackRate = 1.0 + 0.5 * error
    (بين 0.65x و 2.1x)
  - < 0.8s متبقي: preload الجملة التالية

  7. تشغيل الفيديو

  1. تحميل النموذج:
     WhisperModel("small", device="cpu", compute_type="int8")

  2. المعالجة الرئيسية - run_transcription(use_vad, threshold):
     model.transcribe(
       audio_path,
       beam_size=5,
       best_of=5,
       language=None,              // اكتشاف تلقائي
       condition_on_previous_text=True,
       word_timestamps=True,       // توقيت كل كلمة
       no_speech_threshold=0.7,    // قبول المقاطع غير المؤكدة
       log_prob_threshold=-1.2,    // قبول كلام منخفض الثقة
       compression_ratio_threshold=2.8,
       vad_filter=True,
       vad_parameters={
         min_silence_duration_ms: 200,   // صمت 200ms = فاصل جملة
         speech_pad_ms: 400,             // padding حول الكلام
         threshold: 0.20                 // عتبة VAD
       }
     )

  3. المحاولة الأولى: threshold=0.20
  4. Fallback: إذا لم توجد نتائج → threshold=0.10
